Shuffle EXR Passes v1.1

Pr_Suite Python Scripts ›› Nuke Menu ››

Description:

Shuffle EXR Passes is a script to shuffle out selected passes from a Read node which is reading a OpenEXR file. After shuffling out passes Shuffle nodes will be labeled with the layer name they have shuffled.



Operation:

•Select Read node and run script by clicking on Shuffle EXR Passes from Pr_Suite Nuke menu or use the hotkey assigned to it.

•Select passes which you are wanting to shuffle.

•Use 'Select All' to select all passes, use 'Invert Selection' to inverse selection and use 'Clear All' to clear all selection.

•And click on OK.


Special Note:

By default RGBA layer will not be selected.
