Installation

Pr_Suite Introduction ››

•Download Pr_Suite from http://www.nukepedia.com/gizmos/other/pr_suite or http://www.parimalvfx.com/rnd/pr_suite/


•After downloading extract the Pr_Suite package.


•Copy Pr_Suite package to one of the following directories based on your operating system:


Operating System

Installation Directory

Windows

C:\Users\username\.nuke

Mac

Users/username/.nuke

Linux

/home/username/.nuke


•If you are installing Pr_Suite in custom PLUGIN_PATH make sure to copy the whole Pr_Suite package as it is.


•After copying the package add following lines to your root init.py, if you don't have init.py create one by copy and pasting following lines in a text editor and save it as init.py in your .nuke directory:


nuke.pluginAddPath( "./Pr_Suite" )


•Now restart Nuke and you can see Pr_Suite is successfully installed if Pr_Suite icon is visible Nodes menu and Pr_Suite menu is visible in Nuke menu.


Nodes Menu


Nuke Menu


•If Pr_Suite icon is not visible in Nodes menu and Pr_Suite menu in not visible in Nuke menu then your installation is wrong, make sure you have copied the whole package as it is and you have not modified the menu.py and init.py included in the package.


•If you are still not able to install Pr_Suite e-mail at support@parimalvfx.com
