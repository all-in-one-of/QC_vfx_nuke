Pr_Suite Preferences

Pr_Suite Preferences and Nuke Renders Setup ››

Pr_Suite Preferences is great way to customize Pr_Suite. You can change color of all Pr_Suite gizmos from here. And most robust is Plugin Manager with which you can activate only needed gizmos and scripts.


How to use Preferences:

•Open Pr_Suite Preferences from Pr_Suite Nuke Menu.


•Change gizmo color with 'Gizmo Color' knob.


•Check/Uncheck plugins according to your need.


•Save preferences by using 'Save Preferences' knob.


•Or set preferences to default by using 'Set to Default' knob.
