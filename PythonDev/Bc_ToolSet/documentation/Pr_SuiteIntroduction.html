Pr_Suite Introduction

Pr_Suite is a suite of tools for The Foundy Nuke which comprises advanced gizmos, Python scripts and Batch scripts for daily compositing and production needs.


Developed with detailed R&D and production tested so you can rely on Pr_Suite and make it a part of your standard toolset. Pr_Suite has a very informative and up-to-date support in the form of tool integrated help, documentation (which you are currently reading), tutorials, bug reporting and many others.
