Description:

Pr_LBGrain is a luminance based graining tool which is based on default Nuke grain tool but with advanced controls for realistic grain.



Inputs:

Pr_LBGrain has two inputs:

•Main input - This input takes the image on which grain is to be applied.

•Mask input - This is a optional input, if you want to limit the grain to a specific region, feed in matte for limiting region.
