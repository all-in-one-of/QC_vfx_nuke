Multi Knob Values v1.1

Pr_Suite Python Scripts ›› Nuke Menu ››

Description:

Multi Knob Values is a script to change knob values of multiple nodes. You can change values of up to three int/float knobs and three string/enumeration knobs.



Operation:

•Select nodes and run script by clicking on Multi Knob Values from Pr_Suite Nuke menu or use the hotkey assigned to it.

•Enter knob calling name and value for the knobs.

•And click on OK.


Special Note:

Before selecting nodes to change knob values, make sure that same knobs are available in all nodes which you are wanting to change.
