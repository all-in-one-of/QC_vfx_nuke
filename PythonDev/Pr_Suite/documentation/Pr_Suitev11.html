Known Bugs and Workarounds:

•Pr_RGBLuma v1.1:

     Changing options in 'Adjust in' drop down choice for the first time, intensity knob will get a 'nan' value.

To overcome this, change to desired option from 'Adjust in' drop down choice and then set intensity knob to zero before adjusting it. After this you change options from 'Adjust in' drop down choice without any 'nan' value problem in intensity knob.
