Pr_ShuffleMatte v1.1

Description:

Pr_ShuffleMatte is a matte extractor which extracts matte from ID pass (red, green, blue, cyan, magenta, yellow, white and black mattes) and controls it's edges after extracting. Pr_ShuffleMatte can extract multiple mattes and control edges of these mattes separately, the math used to extract the matte can be changed.



Inputs:

Pr_ShuffleMatte has only one input which takes a ID pass (rendered from a 3D application).

•Mask input - This is a optional input, if you want to limit the extracted matte, feed in matte for limiting region.


Special Note:

•Rendering ID pass - While rendering ID pass from a 3D application, render in 16 bit or 32 bit int or float for proper matte extraction. Do not render in 8 bit it will result in improper matte extraction.

•Reading ID pass in Nuke - Make sure you have raw data checked in read node of ID pass in Nuke.

•RGB ID pass - If you are using ID pass which have only RGB colors then using Pr_ShuffleMatte is not recommended, use Nuke's Shuffle node instead to extract mattes.
