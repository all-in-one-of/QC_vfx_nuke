Pr_Suite Nuke Queue Render v1.0

Pr_Suite Nuke Renders (Windows Only) ››

Description:

Pr_Suite Nuke Queue Render is a script to render multiple Nuke files from a folder and its sub-folders in queue.


Operation:

•Right click on folder with Nuke files and go to 'Send to' menu.

•Click on Pr_Suite Nuke Queue Render.

•All Nuke files will get rendered one by one.

•And render log will be generated after all render completes.

•Press 'Pause/Break' button on keyboard to pause render and press 'Enter' to resume render. Close terminal to exit render immediately.


Special Note:

Frame range of every Write node is calculated from Write node's first and last frame, use 'limit to range' in Write node to set frame range. Otherwise it will skip render for that Write node.
